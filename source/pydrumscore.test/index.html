pydrumscore.test.test_base module¶

Contains the base class test class.

class pydrumscore.test.test_base.TestBase(methodName='runTest')¶

Bases: TestCase

Base class for test cases that use a specific song file, export it, and compare the result to a reference data file.

base_test_song(song_name: str) → None¶

Exports the song of the given name and does a diff to compare it to the reference data. Certain divergence are allowed (such as style) while any change in core content fails the test.

Parameters:

song_name – Name of the song for this test case
